= Scripts

Collection of Bash scripts for server management and database administration.

== General
Helper scripts for repository management.

* `Git set permissions windows`: Sets the execute bit for shell scripts under Windows.
[source,powershell]
----
Get-ChildItem -Recurse -Filter *.sh | ForEach-Object { git update-index --chmod=+x $_.FullName }
----

== Linux
General system scripts for Linux.

* `apt-update-checker.sh`: Checks for updates and sends Discord notifications.
* `copy_dir.sh`: Copies directories.
* `discord.sh`: Helper script for Discord error messages.
* `file_backup.sh`: Creates file backups.
* `setup_node_exporter.sh`: Installs the Prometheus Node Exporter.

== MariaDB
Management, backup and monitoring for MariaDB.

* `cleanup_db.sh`: Removes databases.
* `copy_db.sh`: Copies databases.
* `create_db.sh` / `create_user.sh`: Creates DBs and users.
* `reset_password.sh`: Resets user passwords.
* `mariadb_pb.sh`: Physical backup (mariabackup + restic).
* `mariadb_lb.sh`: Logical backup (mariadb-dump + restic).
* `setup_exporter.sh`: Installs the MariaDB Prometheus Exporter.

== MongoDB
Management, backup and monitoring for MongoDB.

* `cleanup_db_user.sh`: Removes database users.
* `create_db_user.sh` / `create_user.sh`: Creates users.
* `reset_password.sh`: Resets user passwords.
* `mongodb_pb.sh`: Physical backup.
* `mongodb_lb.sh`: Logical backup (mongodump).
* `setup_admin.sh`: Initial admin setup.
* `setup_exporter.sh`: Installs the MongoDB Prometheus Exporter.

== PostgreSQL
Management, backup and monitoring for PostgreSQL.

* `change_user_password.sh`: Changes user passwords.
* `cleanup_db.sh`: Removes databases.
* `cleanup_exporter.sh`: Uninstalls the exporter.
* `create_db.sh` / `create_schema.sh` / `create_user.sh`: Initialization scripts.
* `postgresql_pb.sh`: Physical backup.
* `postgresql_lb.sh`: Logical backup (pg_dump).
* `setup_exporter.sh`: Installs the Postgres Prometheus Exporter.
